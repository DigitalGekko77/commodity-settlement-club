# Commodity Settlement Club (CSC)

Товарный клиринг-клуб — расчётная инфраструктура для товарных коридоров.

**Не замена SWIFT. Не крипта. Не обход санкций.** Узкий расчётный слой для товарных потоков, который автоматически начисляет пошлины и работает без единого эмитента.

## Ключевые параметры

| Параметр | Значение |
|----------|----------|
| Комиссия | 0.3% (против 1–3% через банки) |
| Время расчёта | <72 часов (против 7–14 дней) |
| Потенциал бюджета | +$2.5–5 млрд/год при масштабировании |
| Спонсор | НБР БРИКС (Шанхай) |

## Принцип

Каждая страна выпускает товарный токен, привязанный к золоту как расчётной единице (numeraire). Токен гасится реальным экспортным товаром — не золотом и не валютой. Обеспечение — производственный потенциал страны.

## Что это НЕ

- **Не замена SWIFT** — узкий инструмент для отдельных товарных коридоров
- **Не криптовалюта** — корпоративный блокчейн, BFT-консенсус, нет публичной торговли
- **Не обход санкций** — полная интеграция с государственными институтами (ЦБ, Минфин, ФТС, ФНС)
- **Не требует золотых запасов** — золото = расчётная единица, не физический резерв

## Выгоды для государства

- **Банк России** — надзор за ОТП как НКО, ломбардное кредитование под токены
- **Минфин** — автоматическое начисление экспортных пошлин (+$2.5–5 млрд/год)
- **ФТС** — каждый токен = таможенная декларация
- **ФНС** — ОТП как налоговый агент, автоначисление НДС и налога на прибыль
- **Минэкономразвития** — аналитика товарных потоков, veto по номенклатуре
