# Commodity Settlement Club — CTB v4.0

CTB — проект регулируемого многостороннего клирингового клуба для товарных коридоров. Это не параллельная валюта и не готовая промышленная система: v4.0 описывает проверяемую архитектуру пилота, evidence gates и условия, при которых проект может перейти к внедрению.

## Что изменилось в v4.0

- CBU, платёжный актив, цифровая запись требования и электронный товарораспорядительный документ разделены юридически и технически.
- Юридическая окончательность отделена от технического commit.
- Фиксированные комиссия, цикл расчёта, залог и квота погашения больше не считаются универсальными константами: они калибруются по данным коридора.
- Добавлены детерминированный неттинг, лимиты дисбаланса, динамическая маржа и ограниченный default waterfall.
- Описаны selective disclosure, независимые источники Oracle и обязательный человеческий контроль AI.
- Двусторонний коридор признан недостаточным доказательством ключевой гипотезы многостороннего неттинга.

## Готовность

Расчётная design-readiness повышена примерно с 34% до 70%. Production-readiness остаётся условной до получения реальных данных участников, юридических заключений по юрисдикциям, подписанных правил и LOI, калибровки финансовых ресурсов, живых интеграций и независимых испытаний.
